## Data the extension handles

- **Reading**: Use properties (e.g., `tag.title`)
- **View values**: `kebab-case` (e.g., `stepX-my-feature`)
- **Use cases**: Stripe webhooks, external API integrations, custom REST endpoints
- **Main Game Class** (`src/game.h/cpp`) - Unified event processing, context-aware input handling
- **Apexcharts**: Interactive charting library
- **Human-in-the-loop**: Can pause for human approval.
