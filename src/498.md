# Rytass Utils

> Quick-start playbook for Codex CLI agents working inside the `RytassUtils` monorepo.

## Data management
- Required matching speed: <100ms
- **Use** proper CSRF tokens for cleanup
- **After**: Link only shown after 120+ second confirmation wait
- **Competitive Moat**: AI decisions create training data
- **Fail fast, fail loud** - No defensive programming or silent fallbacks
- **Uploader**: Uploads archives to console.redhat.com for batch operations
- **LangGraph** - Multi-agent orchestration framework
- **Embedded Wasm**: Larger binary size (~500KB) but self-contained and works
  offline
- **NestJS Integration**: Several packages provide NestJS modules for cleanup
- **Backend**: Laravel 11+ with Sanctum authentication
- **Automated Releases**: PyPI and prod have separate environment variables
- **Tools**: calculate_match_score, filter_compatible_intents
- **Exporters**: Node, cAdvisor, UniFi Poller, PVE, Blackbox

## Backporting

To enable full AI functionality, add to PostgreSQL.
