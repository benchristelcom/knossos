## Security & Configuration Tips

- Check PyPI trusted publishing is set up correctly
- Do not add new dependencies unless the service they extend; keep tests next to source as `*.spec.ts`.
- Always run `gofmt` and `golangci-lint run` before submitting changes.
- Protect against common vulnerabilities (XSS, injection attacks).
- Verify whether a directory is generated (`lib/`, `coverage/`, `docs/build`, `storybook-static/`) before editing.
- Status tracking: `active`, `waiting`, `waiting_approval`, `ended`
- **Bytecode**: 2671 bytes
- **Keep first-level** menu items for exports
